## O Projeto

* Cadastramento e consulta de Livros.
* Autenticação. Se possível será implementado o [firebase](https://firebase.google.com/).
* MVC.
* Banco de Dados MySQL. Se possível será utilizado PostgreSQL.

## Recursos / funcionalidades do Projeto
* Autenticação usuários
	* Se possível implementação do [firebase](https://firebase.google.com/):
		* Cadastramento pelo próprio usuário;
		* Recuperação de senha;
		* Autentição via e-mail
* CRUD
	* Cadastro
	* Leitura
	* Atualização
	* Exclusão

* API
	* Utilização de API Google Books para cadastramento automático dos livros.
		- Ao informar o ISBN é realizado uma busca no banco de dados do google
